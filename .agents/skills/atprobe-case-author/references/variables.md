# 变量系统（提取 / 引用 / 作用域）

变量让用例能"从响应里取值，在后续步骤里复用"——查询初始值后恢复、跨端口传递 IP、条件跳过等场景都依赖它。

## 变量提取（extract）

从命令响应里用正则提取值，写入用例变量池。每条 `extract` 键 = 一条正则，取**第一个捕获组**作为变量值：

```yaml
steps:
  - command: AT+CEREG?
    assert: { contains: "+CEREG:" }
    extract:
      stat: 'CEREG:\s*\d,(\d+)'        # 提取第一个捕获分组
      lac: 'CEREG:\s*\d+,"([0-9A-F]+)"' # 每条 extract 各取自己第一个捕获分组
```

- 提取失败（正则无匹配）→ 变量值为空字符串，标记为"已提取但空值"（区别于"未定义"）。
- 命名分组 `(?P<name>...)` 不单独处理，统一取第一个捕获分组。
- 要提取多个值就写多条 extract（不同正则或不同变量名）。

## 变量引用：两种语法（场景不同，别混用）

| 场景 | 语法 | 求值方式 | 适用字段 |
|---|---|---|---|
| **字符串内容模板** | `{{变量名}}` | 文本替换，值原样嵌入 | `command`、`data.inline`、`data.file` |
| **条件表达式** | 裸变量名 | 引擎按作用域取值，类型安全比较 | `when`、`poll.until` |

字符串模板——值原样塞进字符串：
```yaml
steps:
  - command: 'AT+CIPSTART="TCP","{{server_ip}}",8080'   # {{}} 文本替换
    assert: { contains: "CONNECT" }
```

条件表达式——裸变量名（条件求值语法详见 `control-flow.md`）：
```yaml
steps:
  - command: 'AT+CGACT=1,1'
    when: 'apn != "" and stat == "1"'   # 裸变量名，引擎求值
```

## 变量作用域：用例级唯一池

- **唯一作用域：用例级变量池。** 变量属于当前用例，生命周期 = 用例生命周期，用例结束即销毁。
- 变量在当前用例的**所有步骤间共享**（setup / steps / teardown，含跨端口步骤）。
- **变量属于用例，不属于端口。** 无论步骤在哪个端口执行，都读写同一个用例变量池。
- 用例之间**不共享**变量，每个用例有独立的变量池。
- 步骤串行执行，前一步骤的 extract 一定在后一步骤开始前完成，变量可见性天然保证，无竞态。
- 同名变量后赋值覆盖（含覆盖参数化参数）。
- **不存在 export 机制，不存在跨用例变量传递。** 跨端口联动通过同一用例内步骤分布在不同端口实现。

### 跨端口共享变量池示例（双模块测试）

服务端建链查 IP，客户端引用该 IP 连接——全程共享一个用例变量池：

```yaml
name: 双模块-TCP通信测试
steps:
  # 步骤1：COM3（服务端）建立服务器并查询IP
  - command: AT+CIPSERVER=1,8080
    port: COM3
    assert: { contains: "OK" }
  - command: AT+CIFSR
    port: COM3
    extract: { server_ip: '(\d+\.\d+\.\d+\.\d+)' }   # 提取IP写入用例变量池

  # 步骤2：COM5（客户端）连接服务端，引用变量池中的IP
  - command: 'AT+CIPSTART="TCP","{{server_ip}}",8080'
    port: COM5
    assert: { contains: "CONNECT" }
```

## 内置变量

| 变量名 | 说明 |
|---|---|
| `{{timestamp}}` | 当前时间戳 |
| `{{port}}` | 当前执行的串口标识 |
| `{{loop_index}}` | 当前循环索引（压测场景，从 1 开始） |

内置变量用于字符串模板。要在条件表达式里引用内置变量，需先 `extract` 成普通变量。

## 环境配置（点号引用，全局只读）

除用例级变量池外，还有一层**全局只读的环境配置**（M7），承载跨用例共享的固化参数（FTP/HTTP 服务器地址、FOTA 版本号等）。

- 引用语法：**点号引用** `{{group.param}}`，如 `{{ftp.host}}`、`{{fota.version_a}}`。
- 查找优先级：**用例级变量池**（extract/parameters，简单名）> **环境配置**（点号名）> 未定义报错。
- 点号名 `{{group.param}}` 只查环境配置，不被 extract 覆盖（保证环境配置全局只读）。
- 用例级动态值用简单名变量另存。
- 环境配置在引擎启动时一次性加载，执行期间只读。
