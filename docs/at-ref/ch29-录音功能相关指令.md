### 29.1 AT+RECMODE — 设置录音模式

设置录音时录音数据的缓存方式。 录制的音频文件保存在模组内部。设置该参数后，掉电不保存。 默认值为0，请在开始录音前设置好录音方式。 文件扩展名不一定是该格式，文件格式以设置第三个参数为准。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  | `AT+RECMODE=<mode>[,<file_` | `<CR><LF>OK<CR><LF>` |
| 设置 | `Or name>[,<format>]]<CR>` | `<CR><LF>ERROR<CR><LF> <CR><LF>+RECMODE: <mode>` |
| 查询 | `AT+RECMODE?<CR>` | `<CR><LF>OK<CR><LF> <CR><LF>+RECMODE: (range of <mode>` |
| 测试 | `AT+RECMODE=?<CR>` | `value),<file_name>,(range of<format> value) <CR><LF>OK<CR><LF>` |

**参数**

| <mode> |  | 0：录本地音频（默认） |
| --- | --- | --- |
|  |  | 1：录voice 下行音频（暂不支持） |
|  |  | 2：录voice 上行音频（暂不支持） |
|  |  | 3：录voice 混合音频 |
|  | <file name> _ | 录音保存文件名，默认从001.pcm 开始保存 |
|  | <format> | 录音格式 |
|  |  | 0：pcm（默认） |
|  |  | 1：wav |
|  |  | 2：mp3（暂不支持） |
|  |  | 3：amr |

**示例**

```
AT+RECMODE=3,voice.pcm,0 设置录制通话混合音频。
OK
AT+RECMODE=0,audio.pcm,0 设置录制本地音频。
OK
AT+RECMODE? 查询录音设置状态。
+RECMODE: 0
OK
AT+RECMODE=? 查询录音设置范围。
+RECMODE: (0-3),<0:pcm,1:wav,2:mp3,3:amr>,(0,1,2,3)
OK
```


### 29.2 AT+RECF — 打开和关闭录音

开始录音和停止录音。 开始录音前，需要设置录音模式。如果没有设置录音文件名称，会从001.pcm 开始保存。 如果模组内部已经存在同名文件，新的录音将会覆盖原来的录音文件。 录音时长最长支持1 分钟，一分钟后没有用AT+RECF=0 停止录音，会上报+RECF:FINISH 停止。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
| 执行 | `AT+RECF=<val>[,<quailty>]<CR>` | `<CR><LF>OK<CR><LF>` |

**参数**

只有通过recmode 指令设置格式为amr 时，quality 参数才有效，其他无效。

| <val> | 0：停止录音 |
| --- | --- |
|  | 1：开始录音 |
| <quality> | 0：low |
|  | 1：medium |
|  | 2：high |
|  | 3：best（默认） |

**示例**

```
AT+RECF=1 开始录音。
OK
AT+RECF=0 停止录音。 录音超时。
OK
AT+RECF=1
OK
+RECF: FINISH
```


### 29.3 AT+RECP — 播放录音文件

播放录音文件。 可以在空闲时播放音频文件；播放文件的格式支持pcm，wav，amr。播放结束后上报+RECP:FINISH。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
| 执行 | `AT+RECP=<sta>[,<file_name0>……[,<file_name15>]]<CR>` | `<CR><LF>OK<CR><LF>` |

**参数**

| <sta> | 音频播放开关。 |
| --- | --- |
|  | 0：停止播放 |
|  | 1：开始播放 |
| <file name> _ | 保存在模组内部的录音文件,最多支持播放16 个wav 文件。 |

**示例**

```
AT+RECP=1,001.wav 开始播放录音文件001.wav。
OK
+RECP:FINISH
AT+RECP=0 停止播放录音。
OK
```


### 29.4 AT+RECFG — 录音文件播放设置

录音文件播放设置。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  |  | `<CR><LF>OK<CR><LF>` |
| 设置 | `AT+RECFG=<type>,<value><CR>` | `Or <CR><LF>ERROR<CR><LF> <CR><LF>+RECFG: <type>,<value>` |
| 查询 | `AT+RECFG?<CR>` | `<CR><LF>OK<CR><LF>` |

**参数**

| <type> | 参数类型 |
| --- | --- |
|  | 0：多文件播放间隔 |
|  | 1~99：预留 |
| <value> | 参数对应的值 |
|  | 当type=0 时，对应值为时间间隔，范围0~10，单位s，默认为0。 |

**示例**

```
AT+RECFG=0, 5 设置播放时间间隔为5s。
OK
AT+RECFG? 查询播放时间间隔为5s。
+RECFG: 0,5
OK
```


### 29.5 AT+AUDPLAY — 播放音频指令

播放音频指令。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  | `AT+AUDPLAY=<type>,<file_name0>[,<file` | `<CR><LF>+AUDPLAY: PLAY FAILED<CR><LF> Or <CR><LF>OK<CR><LF> Or <CR><LF>ERROR<CR><LF> <CR><LF>OK<CR><LF> <CR><LF>+AUDPLAY:PLAY SUCCESSED<CR><LF> Or` |
| 执行 | `_name1>…,<file_nameX>],<mode><CR>` | `<CR><LF>OK<CR><LF> <CR><LF>+AUDPLAY: PLAY STOP Or <CR><LF>ERROR<CR><LF>` |

**参数**

| <type> |  | 播放类型 |
| --- | --- | --- |
|  |  | 0：停止播放 |
|  |  | 1：播放WAV |
|  |  | 2：播放PCM（对应的采样率使用AUDPCMRATE 指令设置） |
|  |  | 3：播放AMR（TBD） |
|  | <file name0~X> _ | 文件名，长度不超过50 个字符，支持多文件播放，最大不超过16 个。 |
| <mode> | <mode> | 播放模式. |
|  |  | 0：非通话时播放，接通电话时需要先停止播放，再接听来电。 |
|  |  | 1：通话时播放，对方能听到本地播放的音频。 |

**示例**

```
AT+FSWF="test.amr",1,1024,10000 向用户盘中写入test.amr 文件，数据内容需按照AMR 文件格式，如：#!AMR…。 播放test.amr 文件，播放完成返回+AUDPLAY: PLAY SUCCESSED。
>
OK
AT+AUDPLAY=3,"test.amr",0
OK
+AUDPLAY: PLAY SUCCESSED
AT+AUDPLAY=2,"test1.amr",0 test1.amr 文件不存在、文件内容格式不正确，提示播放失败。
ERROR
AT+AUDPLAY=0 停止播放。
OK
+AUDPLAY: PLAY STOP
AT+AUDPLAY=0 未播放音频时，返回ERROR。
ERROR
```

### 29.11 AT+REMAUDIO — 设置远程播放开关

设置远程播放开关。 打开远程播放后，本地播放的TTS，音频可在通话时播给对端。 V1.5 基线后的版本支持，有TTS 功能的版本才支持该指令。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  |  | `<CR><LF>OK<CR><LF>` |
| 设置 | `AT+REMAUDIO =<onofff><CR>` | `Or <CR><LF>ERROR<CR><LF> <CR><LF>+REMAUDIO:<onoff><CR><LF>` |
| 查询 | `AT+REMAUDIO?<CR>` | `<CR><LF>OK<CR><LF>` |

**参数**

| <onoff> | 0：关闭远程播放（默认） |
| --- | --- |
|  | 1：打开远程播放 |

**示例**

```
AT+REMAUDIO=0 关闭远程播放 设置远程播放
OK
AT+REMAUDIO=1
OK
AT+REMAUDIO? 查询设置状态
+REMAUDIO: 1
OK
```
