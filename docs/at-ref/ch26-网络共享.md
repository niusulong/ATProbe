### 26.1 AT+NETSHAREMODE — 网络共享模式选择

打开网络共享。 切换共享模式后掉电保存，且重启生效。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  |  | `<CR><LF>OK<CR><LF>` |
| 执行 | `AT+NETSHAREMODE=<share_mode><CR>` | `Or <CR><LF>ERROR<CR><LF> <CR><LF>+NETSHAREMODE:` |
| 查询 | `AT+NETSHAREMODE?` | `<share_mode> <CR><LF>OK<CR><LF>` |

**参数**

| <share mode> _ | 0: RNDIS |
| --- | --- |
|  | 1: ECM |

**示例**

```
AT+NETSHAREMODE=1 设置共享网络模式为ECM。
OK
AT+NETSHAREMODE? 查询网络共享模式。 当前共享模式为ECM。
+NETSHAREMODE: 1
OK
```


### 26.2 AT+NETSHAREACT — 打开网络共享

打开网络共享。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  | `AT+NETSHAREACT=<cid>,<action>` | `<CR><LF>OK<CR><LF>` |
| 设置 | `,<auto>[,<APN>[,<username>[,<pas swd>[,<authtype>[,<ip_family>]]]]]` | `Or <CR><LF>ERROR<CR><LF> <CR><LF>+NETSHAREACT: <stat>,<auto>,<APN>,<username>,<passwd>,<P` |
| 查询 | `AT+NETSHAREACT?` | `DP_type>,<share_mode> <CR><LF>OK<CR><LF> <CR><LF>+NETSHAREACT: (range of supported<cid>),(range of supported<action>),(range of` |
| 测试 | `AT+NETSHAREACT=?` | `supported<auto>),<apn>,<username>,<passwd>, (list of supported<authtype>s,list of supported <ip_family>s) <CR><LF>OK<CR><LF>` |

**参数**

| <cid> | PDP 上下文，取值为2-7。 |
| --- | --- |
| <action> | 执行动作 |
|  | 0：stop |
|  | 1：start |
| <auto> | 选择是否上电后自动打开网络共享。 |
|  | 0：manual |
|  | 1：auto |
| <APN> | 网络参数，APN，字符取值范围为0-99。 |
| <user name> _ | 网络参数，用户名，字符取值范围为0-64。 |
| <passwd> | 网络参数，密码，字符取值范围为0-64。 |
| <auth type> _ | 认证方式 |
|  | 0：NONE |
|  | 1：PAP (默认) |
|  | 2：CHAP |
| <ip family> _ | IP family |
|  | 4：IPv4（默认） |
|  | 6：IPv6 |
|  | 10：IPv4 6 _ |
| <stat> | 0：disconnect |
|  | 1：connect |
| <atuo> |  | 0：manual 1：auto |
| --- | --- | --- |
|  | <err code> _ | 暂不支持 |
|  | <wx stat> _ | 暂不支持 |
|  | <PDP type> _ | IPv4 |
|  | <share mode> _ | RNDIS/ECM |

OK
AT+NETSHAREACT=? 打开网络共享的参数范围。 打开网络共享的参数范围。
+NETSHAREACT: (2-7),(0-1),(0-
1),"apn","user","passwd",(0-2),(4,6,10)

AT+NETSHAREACT? 查询网络共享状态。 查询网络共享状态。
+NETSHAREACT: 1,0,"ctnet","card","card","IPV4",RNDIS
