### 33.1 AT+MYPIPECLOUDOPT — 管道云开启关闭

管道云开启与关闭

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  | `AT+MYPIPECLOUDOPT=<OPT>,<SERVIC` | `<CR><LF>OK<CR><LF>` |
| 执行 | `E_TYPE><CR>` | `Or <CR><LF>ERROR<CR><LF>` |

**参数**

| OPT SERVICE TYPE _ | 0：关闭管道云1：开启管道云 |
| --- | --- |
|  | 2：有方管道云 |

**示例**

```
AT+MYPIPECLOUDOPT=1,2 OK 开启有方管道云
AT+MYPIPECLOUDOPT=0,2 关闭有方管道云
OK
```


### 33.2 AT+MYPIPECLOUDCFG — 管道云服务器参数设置

服务器参数设置，查询。支持掉电保存。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  | `>,<SERVER_PORT><CR>` | `C 支持的band 设置参见附录I URC 命令 Or <CR><LF>ERROR<CR><LF> <CR><LF>+MYPIPECLOUDCFG:` |
| 查询 | `AT+MYPIPECLOUDCFG? <CR>` | `<SERVER_URL>,<SERVER_PORT><CR> <LF>OK<CR><LF>` |
| 执行 | `AT+MYPIPECLOUDCFG=<SERVER_URL` | `<CR><LF>OK<CR><LF>` |

**参数**

| SERVER URL _ SERVER PORT _ | 服务器域名或者IP 地址 |
| --- | --- |
|  | 服务器的端口 |

**示例**

```
AT+MYPIPECLOUDCFG=”219.144.245.178”,1883 设置服务器参数
AT+MYPIPECLOUDCFG=”iot.neoway.com,1883
AT+MYPIPECLOUDCFG?
+MYPIPECLOUDCFG: "219.144.245.178",1883
OK
```
