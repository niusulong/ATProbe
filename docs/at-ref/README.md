# N58 AT 命令手册 — Markdown 版

> 源文档：《N58 AT 命令手册 v2.0》（Neoway N58 LTE Cat.1 模组，2024-12-03），共 372 页、33 章 + 附录 A–J。
> 本目录由 PDF 提取并按功能模块拆分为 Markdown，供 ATProbe 工具查询与编写测试用例参考。

## 拆分方式

- 按原书章节一对一拆分（33 章），每章一个 `.md` 文件。
- 文件命名：`ch{章节号两位}-{中文标题}.md`，如 `ch04-通用AT指令.md`。
- 附录单列：`附录{字母}-{标题}.md`。
- 每条命令统一结构：**描述 → 命令格式表（类型/命令/响应格式）→ 参数表 → 示例代码块**。
- 命令/响应中的 `<CR>`/`<CR><LF>` 控制符原样保留；参数占位符用 `<尖括号>` 标记。

## 转换工具与方法

按 PDF 技能规范执行：

- **提取**：pdfplumber（文本骨架）+ pymupdf `get_text("dict")`（带坐标 span）。
- **表格重建**：手册「命令格式表」主体无边框，用 span 坐标按 x 分三列桶重建（类型 x<110 / 命令 110-285 / 响应 x>285）；「参数表」有完整边框，用 `find_tables()` 几何提取并去合并单元格噪声列。
- **校验**：pymupdf 渲染重点页 PNG 做视觉对照，第 4 章手工精校。
- 转换脚本：`tools/build_at_ref.py`（处理 ch05–ch33，可重复运行）。

## 已知限制（自动转换）

- **第 4 章** 为手工精校版，表格质量最高；其余章节为坐标驱动自动转换。
- **命令格式表**：少数命令首行可能出现空的类型单元格（响应行排版先于类型行）。
- **复杂参数表**：含大量合并单元格的表（如 AT+NETDMSG、AT+COPS）部分说明行可能未完美对齐，参数名/取值仍在但顺序偶有错位，需结合原 PDF 核对。
- 如需用于正式发布，建议对 ch05 之后各章的关键命令做一次人工复核。
