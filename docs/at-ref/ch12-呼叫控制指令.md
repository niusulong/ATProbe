### 12.1 ATD — 呼叫指令

语音呼叫，其字符串由数字和修饰符组成。 语音呼叫：使用ATD<dial string>;指令。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  | `ATD<dial string>[;]<CR> ` | `<CR><LF>OK<CR><LF>` |
| 执行 | `ATD><n>;<CR>  ATD>"name";<CR> ` | `<CR><LF><result><CR><LF>` |

**参数**

| <dial string> | 电话号码 |
| --- | --- |
| <n> | 当前电话本中的记录号。 |
| name | 当前电话本中的名称。 |
| <result> | CONNECT：对端摘机 |
|  | NO CARRIER：通话中，对端挂机或者对端未接电话直接挂断。 |

**示例**

```
ATD10010; 拨打电话。
OK
CONNECT 对方应答。
ATD>4; 使用当前电话本中的记录号拨打电话。 使用当前电话本中的记录号拨打电话。
OK
CONNECT
ATH
OK
ATD>"Comneon"; 使用当前电话本中的名称来拨打电话，对方接通后主动挂断。
OK
CONNECT
NO CARRIER
```


### 12.2 ATA — 呼叫应答

应答来电，建立通话链接。 当回码显示RING 或+CRING 时即表示有来电。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  |  | `<CR><LF>OK<CR><LF>` |
| 执行 | `ATA<CR>` | `Or <CR><LF>BUSY<CR><LF>` |

**参数**

| 参数 | 说明 |
| --- | --- |
|  | 无 |

**示例**

```
ATA 应答来电
OK
```

### 12.4 AT+CLIP — 来电显示

设置是否开启模组的来电显示主动上报功能。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  |  | `<CR><LF>OK<CR><LF>` |
| 设置 | `AT+CLIP=<n><CR>` | `Or <CR><LF>ERROR<CR><LF> <CR><LF>+CLIP: <n>,<m>` |
| 查询 | `AT+CLIP?<CR>` | `<CR><LF>OK<CR><LF> <CR><LF>+CLIP: (value range of <n>)` |
| 测试 | `AT+CLIP=?<CR>` | `<CR><LF>OK<CR><LF>` |

**主动上报**

<CR><LF>+CLIP: <phone number>,<tosca>

**参数**

| <n> | 0：关闭来电显示主动上报（默认） |
| --- | --- |
|  | 1：开启来电显示主动上报 |
| <m> | 0：CLIP not provisioned |
|  | 1：CLIP provisioned |
|  | 2：unknown (e.g. no network, etc.) |
| <tosca> | 短信中心号码格式 |

**示例**

```
AT+CLIP=1 使能来电显示主动上报功能。
OK
RING 有电话呼入，呼入号码为：136********。
+CLIP: "136********",161,"",0,"",0
AT+CLIP? 查询模组当前来电显示主动上报功能设置。
+CLIP: 1,1
OK
AT+CLIP=?
+CLIP: (0,1)
OK
```


### 12.5 ATS0 — 自动应答

控制模组的自动应答模式。 若ATS0=0，自动应答功能不使能；若ATS0 非零，模组将在振铃设置次数后自动接听来电。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  |  | `<CR><LF>OK<CR><LF>` |
| 设置 | `ATS0=<value><CR>` | `Or <CR><LF>ERROR<CR><LF> <CR><LF><value>` |
| 查询 | `ATS0?<CR>` | `<CR><LF>OK<CR><LF>` |

**参数**

| <value> | 取值范围0~255，默认值为0 |
| --- | --- |

**示例**

```
ATS0=1 设置来电响铃一次后自动应答。
OK
ATS0? 查询模组自动应答功能设置状态。
1
OK
```


### 12.6 AT+CLVL — 通话音量

设置模组通话音量等级，在通话前、通话中设置均有效。 该指令设置后参数掉电不保存。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  |  | `<CR><LF>OK<CR><LF>` |
| 设置 | `AT+CLVL=<level><CR>` | `Or <CR><LF>ERROR<CR><LF> <CR><LF>+CLVL: <level>` |
| 查询 | `AT+CLVL?<CR>` | `<CR><LF>OK<CR><LF> <CR><LF>+CLVL: (range of <level> value)` |
| 测试 | `AT+CLVL=?<CR>` | `<CR><LF>OK<CR><LF>` |

**参数**

| <level> | 整型，取值范围0~100，默认值为60 |
| --- | --- |
|  | 最小等级为0 |
|  | 最大等级为100 |

**示例**

```
AT+CLVL=4 设置通话音量等级为4。
OK
AT+CLVL? 查询模组当前通话音量等级。
+CLVL: 4
OK
AT+CLVL=? 查询模组允许设置的通话音量等级。
+CLVL: (0-100)
OK
```


### 12.7 AT+CMUT — 静音控制

对语音通话进行静音控制，在通话中设置有效。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  |  | `<CR><LF>+CMUT: (value range of <m>)` |
| 测试 | `AT+CMUT=?<CR>` | `<CR><LF>OK<CR><LF> <CR><LF>OK<CR><LF>` |
| 设置 | `AT+CMUT=<m><CR> Or` | `<CR><LF>ERROR<CR><LF> <CR><LF>+CMUT: <m>` |
| 查询 | `AT+CMUT?<CR>` | `<CR><LF>OK<CR><LF>` |

**参数**

| <m> | 0：关闭输入设备静音（默认值） |
| --- | --- |
|  | 1：打开输入设备静音 |

**示例**

```
AT+CMUT=0 关闭模组静音功能。
OK

OK
AT+CMUT? 查询模组静音功能状态。
+CMUT: 0
OK
AT+CMUT=? 查询静音功能状态的取值范围。
+CMUT: (0-1)
OK
```


### 12.8 AT+CLCC — 列出当前的呼叫

列出当前呼叫列表。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  | `<CR><LF>[+CLCC: <CR><LF>[+CLCC:` | `<idx>,<dir>,<stat>,<mode>,<mpty>,[<number>,<type>,[<alpha>]]]` |
| 执行 | `AT+CLCC<CR> <CR><LF>[...]]] <CR><LF>OK<CR><LF>` | `<idx>,<dir>,<stat>,<mode>,<mpty>,[<number>,<type>,[<alpha>]]]` |

**参数**

| <idx> | 列表序号 |
| --- | --- |
| <dir> | 0 呼出 |
|  | 1 呼入 |
| <stat> | 通话状态 |
|  | 0 活跃 |
| --- | --- |
|  | 1 举行 |
|  | 2 拨号 |
|  | 3 提醒 |
|  | 4 来电 |
|  | 5 等待 |
| <mode> | 承载服务 |
|  | 0 语音 |
|  | 1 数据 |
|  | 2 传真 |
| <mpty> | 是否为多方通话 |
|  | 0 呼叫不是多方通话之一 |
|  | 1 电话是多方呼叫方之一 |
| <number> | 呼叫号码 |
| <type> | 号码类型 |
|  | 145 国际接入号码，包含“+” |
|  | 129 其他号码 |
| <alpha> | 呼叫号码在电话本中对应的名称，字符格式根据+CSCS 中的设置。 |

**示例**

```
AT+CLCC 呼入
+CLCC: 1,1,4,0,0,"13596722590",129
OK
AT+CLCC 呼出
+CLCC: 1,0,2,0,0,"13596722590",129
OK
```


### 12.9 AT+SETVOLTE — 设置VoLTE 开关

设置VoLTE 开关。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  |  | `<CR><LF>OK<CR><LF>` |
| 设置 | `AT+SETVOLTE=<onoff><CR>` | `Or <CR><LF>ERROR<CR><LF> <CR><LF>+SETVOLTE: <onoff>` |
| 查询 | `AT+SETVOLTE?` | `<CR><LF>OK<CR><LF>` |

**参数**

| <onoff> | 0：关闭VoLTE |
| --- | --- |
|  | 1：打开VoLTE（默认） |

**示例**

```
OK
```
