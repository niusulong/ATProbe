### 27.1 AT+FLOWCALC — 统计数据总流量

用于统计通过模组收发的字节数。 该指令用于统计当前通过模组的总流量，单位byte。 仅支持外部协议栈拨号时的流量统计。 V003 及之后的版本支持。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  |  | `<CR><LF>+FLOWCALC: <rx_count>,<tx_count>` |
| 查询 | `AT+FLOWCALC?<CR>` | `<CR><LF>OK<CR><LF>` |

**参数**

| <rx count> _ | 接收数据总量，单位byte。 |
| --- | --- |
| <tx count> _ | 发送数据总量，单位byte。 |

**示例**

```
AT+FLOWCALC? 查询模组收发数据总量。
+FLOWCALC: 1355,1260
OK
AT+FLOWCALC? 查询模组收发数据总量。
+FLOWCALC: 0,0
OK
```


### 27.2 AT+RATECALC — 统计瞬时数据流量

用于统计瞬间通过模组收发的字节数。 该指令用于统计瞬间(100ms)通过模组的流量，单位Byte。 仅支持外部协议栈拨号时的流量统计。 V003 及之后的版本支持。

**命令格式**

| 类型 | 命令 | 响应格式 |
| --- | --- | --- |
|  |  | `<CR><LF>+RATECALC: <rx_count>,<tx_count>` |
| 执行 | `AT+RATECALC?<CR>` | `<CR><LF>OK<CR><LF>` |

**参数**

| <rx count> _ | 接收数据总量，单位B。 |
| --- | --- |
| <tx count> _ | 发送数据总量，单位B。 |

**示例**

```
AT+RATECALC? 查询模组100ms 收发数据总量。
+RATECALC: 1355,1260
OK
AT+RATECALC? 查询模组100ms 收发数据总量。
+RATECALC: 0,0
OK
```
